The QIIME tutorials illustrate how to use various features of QIIME. We recommend that all users begin with either the QIIME Illumina Overview Tutorial or the QIIME 454 Overview Tutorial. These tutorials take the user through a full analysis of sequencing data. After you’ve begun analyzing your own data, you’ll want to move on to the special-purpose tutorials as needed.

The QIIME development team is very interested in incorporating community-contributed tutorials. Please see the contributing to QIIME guidelines if you’re interested in contributing.
